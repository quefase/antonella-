Responsividade: é a capacidade de um site ou aplicativo se ajustar automaticamente a diferentes tamanhos de tela, dispositivos e resoluções, garantindo uma boa experiência de usuário, independentemente do dispositivo utilizado (como desktop, tablet ou smartphone).

A propriedade alt sereve para informar o que está contido naquela tag, caso seu conteúdo não possa ser exibido corretamente.

A tag input é usada em HTML para criar campos de entrada em formulários, permitindo que os usuários insiram dados. Ela pode ser usada para diversos tipos de entrada, como texto, senha, e-mail, número, checkbox, rádio, entre outros.

A tag label é usada em HTML para associar um texto descritivo a um elemento de entrada (input), tornando a interação do usuário mais acessível e intuitiva.Ela melhora a usabilidade dos formulários ao permitir que o usuário clique no texto do rótulo para ativar o campo correspondente.
